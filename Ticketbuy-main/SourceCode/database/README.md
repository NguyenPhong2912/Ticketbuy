# Database

## Kết nối Database

File `db_connection.php` chứa thông tin kết nối database:
- Host: `127.0.0.1`
- Database: `bus_booking_db`
- User: `root`
- Password: `admin123`

## Export Database từ phpMyAdmin

1. Mở phpMyAdmin: `http://localhost/phpmyadmin`
2. Chọn database `bus_booking_db` ở sidebar bên trái
3. Click tab **Export** ở menu trên
4. Chọn phương thức: **Quick** hoặc **Custom**
5. Click **Go** để tải file SQL

File SQL sẽ chứa toàn bộ cấu trúc và dữ liệu của database.
